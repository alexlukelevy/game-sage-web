# Game Sage

A simplistic React front-end to the Game Sage Elasticsearch data.

![screenshot](./screenshot.png)

### Running
If you want to run the front-end yourself, ensure you have first followed the instructions for running the Game Sage [back-end](https://github.com/alexlukelevy/game-sage).

Then start a local server using

```sh
npm run start
```

or


```sh
yarn start
```
